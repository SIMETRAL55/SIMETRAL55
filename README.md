## ⭐ Featured projects

### MeetBot — **Meeting audio → searchable knowledge**
**Repo:** `SIMETRAL55/MeetBot`  
**Stack:** Whisper · Pyannote · sentence-transformers · ChromaDB · LangChain · NiceGUI · Docker  
**One-liner:** Web app that turns meetings into searchable, speaker-attributed knowledge (transcribe → diarize → embed → retrieve).  
**Highlights:** multilingual ASR, speaker diarization, local LLM summarization, Dockerized CPU/CUDA variants.  
👉 https://github.com/SIMETRAL55/MeetBot

---

### Gazebo-Motion-Capture — **Thermal mocap in sim**
**Repo:** `SIMETRAL55/Gazebo-Motion-Capture`  
**Stack:** ROS 2 · Gazebo · OpenCV · Python (rclpy) · colcon · ament  
**One-liner:** Simulate thermal cameras in Gazebo and triangulate 3D targets using synchronized multi-view thermal detection.  
**Highlights:** ROS↔Gazebo bridge, multi-view triangulation, RViz visualization markers.  
👉 https://github.com/SIMETRAL55/Gazebo-Motion-Capture
